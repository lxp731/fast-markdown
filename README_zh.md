# fast-markdown.vim

一款轻量级 Vim 插件，专为 Markdown 高速编辑而生。只需几次按键即可插入标题、行内格式、链接、图片和表格——从此告别手敲重复语法。

## 功能一览

- **标题** — 插入 `#` 至 `######`，自动附带换行
- **行内格式** — 粗体、斜体、删除线、行内代码
- **链接与图片** — 插入 `[]()` 和 `![]()`，带占位标记
- **代码块** — 围栏代码块，预留语言标注位置
- **表格** — 一键生成 N 列表格（表头 + 分隔行 + 数据行），支持追加新行
- **占位跳转** — 使用 `,f` 跳至下一个编辑点（借鉴 vim-latex 的 `<++>` 设计）

## 环境要求

- Vim 8.0+ 或 Neovim
- vimrc 中开启 `filetype plugin on`（绝大多数发行版默认开启）

## 安装

### 快速安装（curl）

```bash
curl -o $HOME/.vim/fast-markdown.vim \
     --create-dirs \
     https://raw.githubusercontent.com/lxp731/fast-markdown/refs/heads/main/plugin/fast-markdown.vim
```

然后在 `~/.vimrc` 中加入：

```vim
source $HOME/.vim/fast-markdown.vim
```

### 插件管理器

**vim-plug：**

```vim
Plug 'lxp731/fast-markdown'
```

**lazy.nvim（Neovim）：**

```lua
{ 'lxp731/fast-markdown' }
```

## 快捷键

所有快捷键仅在 Markdown 文件（`.md`、`.markdown`）的**插入模式**下生效。

> **提示：** `<++>` 是占位标记。插入代码片段后，光标会停留在第一个待编辑位置。输入内容后按 `,f` 即可跳至下一个占位处——类似于现代编辑器中的 Tab 键。

### 标题

| 快捷键 | 生成结果                |
|--------|----------------------|
| `,1`   | `# █` + 换行           |
| `,2`   | `## █` + 换行          |
| `,3`   | `### █` + 换行         |
| `,4`   | `#### █` + 换行        |
| `,5`   | `##### █` + 换行       |
| `,6`   | `###### █` + 换行      |

*（█ = 插入后的光标位置）*

### 行内格式

| 快捷键 | 生成结果   |
|--------|----------|
| `,b`   | `**█**`  |
| `,i`   | `*█*`    |
| `,s`   | `~~█~~`  |
| `,d`   | `` `█` `` |

### 链接、图片与代码块

| 快捷键 | 生成结果                                              |
|--------|----------------------------------------------------|
| `,a`   | `[█](<++>)`                                        |
| `,p`   | `![█](<++>)`                                       |
| `,c`   | `` ``` `` + 换行 + `<++>` + 换行 + `` ``` ``        |

### 表格

| 快捷键 | 功能                                |
|--------|-----------------------------------|
| `,t2`  | 插入 **2 列** 表格（表头 + 分隔行 + 数据行） |
| `,t3`  | 插入 **3 列** 表格                    |
| `,t4`  | 插入 **4 列** 表格                    |
| `,t5`  | 插入 **5 列** 表格                    |
| `,t6`  | 插入 **6 列** 表格                    |
| `,l2`  | 追加一行 **2 列** 数据行                 |
| `,l3`  | 追加一行 **3 列** 数据行                 |
| `,l4`  | 追加一行 **4 列** 数据行                 |
| `,l5`  | 追加一行 **5 列** 数据行                 |
| `,l6`  | 追加一行 **6 列** 数据行                 |

**示例** — 按下 `,t3` 生成：

```
|<++>|<++>|<++>|
|:----:|:----:|:----:|
|<++>|<++>|<++>|
```

按下 `,l3` 在当前行下方追加一行：

```
|<++>|<++>|<++>|
```

### 导航与其他

| 快捷键    | 功能                        |
|----------|---------------------------|
| `,f`     | 跳至下一个 `<++>` 占位符        |
| `<C-s>`  | 保存文件（`:w`）              |

## `<++>` 占位机制详解

1. 按下 `,a` 插入链接，得到 `[](<++>)`，光标位于方括号中间。
2. 输入链接文本，然后按 `,f`——光标跳至 `<++>` 处，占位符自动删除，继续输入 URL。
3. 部分片段末尾还有一个冗余的 `<++>`（如 `,a`、`,p`）——再按一次 `,f` 即可到达，继续编辑后续内容。

**不要手动删除 `<++>`。** 它是你的导航系统。用 `,f` 跳到它上面，开始输入时它会自动消失。
